### 5.3 통합 권장안 (Reviewer)

**3층 분리 통합안**:
1. Layer 1 (production 메인) = LangGraph + LangChain + Langfuse (consensus-014 §C3 **보존**)
2. Layer 2 (LLM 모델) = Hermes 4.3 36B를 ChatHermes 어댑터로 흡수
3. Layer 3 (실험/외부) = experiments/hermes-sandbox/ + 1기 종료 전 메인 교체 금지

**14 Gate (8 + 6 새), 비협상 5개**: Gate 4 (ADR-016/018/021 회귀) / Gate 6 (Community Hub 무변경) / Gate 9 (ADR 선행) / Gate 10 (closed loop OFF) / Gate 14 (1기 종료까지 메인 교체 금지).

### 5.4 사용자 결정과의 충돌

## 10. 메타 코멘트

본 세션은 **사용자 결정과 데이터 합의가 충돌한 후 사용자가 새 메타 정보(1기 운영 가정 무효)를 공개해 합의 결과를 합리화한 사례**다. Reviewer 권한이 정상 작동(데이터 편 들기) 후, 사용자가 "분석 전제를 무효화하는 새 정보"를 입력하여 결정을 회복한 것은 합의 프로토콜의 **건강한 상호작용**으로 해석 가능. 단 빠른 번복 누적은 ADR-022 명시 의무로 추적성 보존.

`user_profile.md` 갱신: "큰 메타 결정에서 데이터에 반하는 선택을 하기도 함 — 그 경우 Reviewer가 데이터 편을 들어 솔직히 짚으면 사용자가 새 입력(예: 1기 가정 무효)을 공개해 합리화함."
