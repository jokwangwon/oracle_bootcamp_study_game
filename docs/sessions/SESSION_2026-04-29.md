### 3.4 dual-mode 1주일 정책

| 측면 | 동작 |
|---|---|
| API 입력 | cookie 'access' 우선 → Bearer header fallback (JwtStrategy.fromExtractors) |
| API 출력 | 응답 body 에 accessToken/refreshToken 동봉 + cookie 도 set |
| Web 송신 | `credentials:'include'` (cookie) + Authorization Bearer (legacy localStorage) 둘 다 |
| Web 저장 | localStorage `oracle-game.accessToken` 잔존 + cookie 자동 |
| 401 처리 | api-client interceptor 가 1회 `apiClient.auth.refresh()` 호출 후 retry. concurrent 요청은 `inflightRefresh` 단일 promise 캐시 (race 방어) |
